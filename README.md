# Awareness-Bot2
The Cyber Security Awareness Bot is an interactive, question-and-answer digital assistant. Its primary goal is to spread digital literacy and teach users how to stay safe online.
Detailed Topic Coverage
The bot is programmed with a comprehensive knowledge base covering seven critical domains of cybersecurity:

1. Password Security
Concepts Covered: Password length, complexity, and uniqueness.

2. Phishing Awareness
Concepts Covered: Email, SMS (Smishing), and phone scams.
Teaches users how to spot red flags like urgent language, generic greetings, mismatched email addresses, and suspicious links.

3. Online Scams
Concepts Covered: Fake tech support, online shopping fraud, and identity theft traps.

Bot Insights: Helps users identify "too good to be true" offers and teaches them to verify sellers before making online payments.

4. Data Privacy
Concepts Covered: Personal Data Protection and Social Media Privacy.

Bot Insights: Guides users on how to limit the amount of personal information they share publicly and how to check app permissions on their devices.

5. Safe Browsing
Concepts Covered: Secure connections and web safety.

Bot Insights: Explains the difference between HTTP and HTTPS, the risks of using public Wi-Fi networks without a VPN, and how to avoid malicious downloads.

6. Malware Education
Concepts Covered: Viruses, Ransomware, and Spyware.

Bot Insights: Defines how malware infects a device, the warning signs of an infected system, and the critical importance of keeping software updated.

7. Social Engineering
Concepts Covered: Psychological manipulation and impersonation.

Bot Insights: Educates users on how cybercriminals trick people into giving away secrets (like pretending to be a bank or a coworker) and how to safely verify a sender's true identity.

Key Features
Instant Answers: Provides real-time cybersecurity advice without making users search the web.

Plain Language: Translates complex technical security concepts into simple, easy-to-understand tips.

Proactive Checklists: Gives users clear steps to take if they think they are currently being targeted by a scam.

Project Objectives & Learning Outcomes

This project was developed to demonstrate:

*Application of conversational AI in addressing real-world security challenges.

*Understanding of core cybersecurity principles and user education methods.

*Secure coding practices, including the safe management of API keys and environment variables.
